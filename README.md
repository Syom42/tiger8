# IronLog 💪 - מדריך התקנה

## הורדה והפעלה

### שלב 1 - הורד את הקובץ
- הורד את קובץ `index.html` למכשיר שלך

### שלב 2 - פתח בדפדפן
- **iPhone**: פתח בספארי (Safari)
- **אנדרואיד**: פתח בכרום (Chrome)

### שלב 3 - התקן על המסך הראשי

**אייפון (iOS):**
1. לחץ על כפתור השיתוף ⬆️
2. גלול ובחר "הוסף למסך הבית"
3. לחץ "הוסף"

**אנדרואיד:**
1. לחץ על שלוש הנקודות ⋮ בכרום
2. בחר "הוסף למסך הבית"
3. לחץ "הוסף"

## שימוש

1. **הרשמה** - לחץ על "פרופיל" והכנס שם
2. **תוכנית שבועית** - הגדר מה עושים כל יום
3. **אימון חדש** - לחץ "התחל אימון חדש" בבית
4. **תרגיל** - הכנס משקל + חזרות, לחץ ✓ לאחר כל סט
5. **טיימר** - נפתח אוטומטית אחרי כל סט ✓
6. **סיום** - לחץ "סיים אימון" בסוף
